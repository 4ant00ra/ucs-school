Note: ADMX templates need to be copied to the Policies/PolicyDefinitions folder
in the sysvol. The Windows Group Policy Management Console then prefers these
to the ADMX templates on the local machine. Thus, this requires that all the
basic Windows ADMX templates need to be copied to the server folder as well
(once).

Administrative Templates (ADMX) for Windows Server 2008 R2 and Windows 7:
http://www.microsoft.com/en-us/download/details.aspx?id=6243

There are also updated standard Windows ADMX templates updated for Windows 8 / Windows Server 2012:
http://www.microsoft.com/de-de/download/details.aspx?id=36991
