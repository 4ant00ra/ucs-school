The files in this directory are based on FirefoxADM_0.5.9.4 (http://sourceforge.net/projects/firefoxadm/).
Adjustments by Univention include conversion of the ADM files to ADMX format and update of the VBS scripts
firefox_startup.vbs and firefox_shutdown.vbs.

See e.g. http://docs.univention.de/en/ucsschool.html for installation and configuration examples.
See also the README.txt in the examples/GPO directory for general instructions.
